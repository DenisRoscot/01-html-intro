Das Web verstehen

Das World Wide Web ist ein riesiges System von verknüpften Dokumenten, die über das Internet abgerufen werden.

Wie das Internet funktioniert

Das Internet ist die physische Infrastruktur (Kabel, Server, Router), während das Web nur ein Dienst ist, der darauf läuft.

Die Rolle von Browsern

Browser wie Chrome, Firefox oder Safari übersetzen Code in visuelle Oberflächen, die wir lesen können.

Grundbausteine einer Webseite

Jede Seite, die du besuchst, besteht aus drei Hauptkomponenten:

  • HTML: Die Struktur und der Inhalt.
  • CSS: Das Design und Layout.
  • JavaScript: Die Interaktivität und Logik.
